Modul LLPDU8P01

Hier befindet sich das Repository für das FHEM Modul für die LogiLink PDU8P01 und da vermutlich baugleich für die Intellinet 163682.

Installation

Modul

Der Inhalt kann mit dem Befehl:
update all https://markusfeist.github.io/FhemLogiLinkPDU8P01/repository/<Name des Repositories>/controls_logilinkpdu8p01.txt
in FHEM bezogen werden. Als Namen stehen zur Verfügung:

  • master: Aktuelle Version (getestet)
  • develop: Aktuelle Entwicklung (ungeprüft)
  • <TAG>: Die getagte Releaseversion.

Einrichtung

Das Modul legt man dann im FHEM mit dem Befehl:
define <name> LLPDU8P01 <IP/Hostname> <Pollintervall> <Username> <Password>
an. Also z.B.:
define PDU1 LLPDU8P01 192.168.1.3 30 admin admin

Module für Steckdosen

Module für die Steckdosen legt man am besten mit dem Set-Befehl autocreate an.
